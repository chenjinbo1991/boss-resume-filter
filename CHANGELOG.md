# 更新日志

## v2.8.10 — 界面体验打磨

### 体验优化
- **界面字体更统一**：表单标签、按钮、下拉框等常用控件统一使用同一套字体，减少不同区域字体大小不一致的情况
- **候选人明细弹窗优化**：点击统计数字弹出的明细窗口高度增加，方便查看更多候选人；"已打招呼"人数用绿色醒目显示，一眼区分
- **筛选进度条更美观**：运行时进度条高度与右侧百分比文字对齐，不再高低不齐
- **AI 评估勾选框加大**：运行控制页"启用 AI 辅助评估"勾选框加大，鼠标更容易点中
- **表格表头统一加粗**：岗位配置的技能关键词列表、筛选结果列表、数据统计的岗位明细表，三处表头统一为加粗样式
- **侧边栏更紧凑**：左侧导航栏字体缩小，同屏可显示更多导航项，减少滚动
- **需求输入框更醒目**：岗位配置页"粘贴招聘需求文档"输入框使用深蓝色文字，与其他输入框区分更明显
- **高分屏窗口居中修复**：修复 4K 等高缩放显示器上主窗口启动时位置偏上或偏左的问题

### 问题修复
- **Apple Silicon Mac 字体偏小修复**：修复 M1/M2/M3/M4 芯片 Mac 上界面字体明显偏小的问题；Intel Mac 不受影响，Windows 不受影响

## v2.8.9 — 更新与数据稳定性补丁

### 新增功能
- **国内备用更新源**：新增 Gitee 更新检测，国内用户无需 VPN 也能快速检测新版本；Gitee 失败时自动回退 GitHub
- **国内下载资源补齐**：发布时同步提供 Windows、Mac、配置文件和说明文档的国内下载入口，减少用户下载失败的情况

### 体验优化
- **4K 高 DPI 显示优化**：在 4K 大屏高缩放环境下明显等比例缩小界面，减少窗口和控件偏大的感觉
- **候选人数据保存更稳**：删除候选人、清空候选人、更新打招呼状态时统一使用安全保存流程，减少异常中断导致数据损坏的风险
- **发布检查更完整**：发布前自动检查更新日志、待办事项、敏感文件、源码编译、稳定单元测试和导入烟测，减少漏更和误发
- **AI 评估回归输出更清爽**：测试输出减少无关运行日志，回归结果更容易快速确认

### 问题修复
- **候选人备份一致性修复**：修复部分图形界面操作未经过备份保存流程的问题，现在候选人数据变更会保持一致的备份和恢复能力
- **更新日志明细换行修复**：修复版本历史页个别明细行换行后只显示圆点、正文掉到下一行的问题
- **下载失败回退修复**：修复国内下载源不可用时无法自动切换备用下载地址的问题
- **跨平台发布补齐修复**：修复本地只发布单个平台产物后，另一个平台安装包不能自动补齐的问题
- **发布流程稳定性修复**：修复更新清单生成后发布流程可能被错误中断的问题
