# AI 驱动的人力资源池智能招聘实践

## 打开方式

双击 `index.html`，建议使用 Chrome 或 Edge 全屏演示。

如果浏览器限制本地资源，可在项目根目录运行：

```powershell
python -m http.server 8765 --bind 127.0.0.1 --directory "docs\高伟达AI智能招聘演示"
```

然后访问：

```text
http://127.0.0.1:8765/index.html
```

## 演示控制

- `→` / `←`：前后翻页
- 空格：下一页；在流程页中逐步点亮流程
- `ESC`：打开或关闭 15 页索引
- `Home` / `End`：跳到首尾页
- `B`：切换动态与静态模式

## 内容说明

- 共 15 页，采用“电子杂志 × 电子墨水”风格和“靛蓝瓷”主题。
- 岗位配置、筛选结果、AI 评估和统计看板均来自当前真实系统。
- 候选人姓名、编号、公司、学校等个人信息已替换。
- 统计数据使用当前真实业务快照，不包含虚构的效率或准确率指标。
