# 学历证书核验助手 — 操作说明书

## 1. 工具简介

学历证书核验助手是一款独立运行的桌面工具，用于批量核验候选人毕业证书信息。核心流程：

1. **导入**毕业证书图片或 PDF
2. **AI 识别**姓名、证书编号、学校、专业
3. **自动填写**学信网查询页面并提交
4. **HR 人工完成**手机扫码验证最终结果

工具内置 AI 模型和 API Key，开箱即用，无需额外配置。

---

## 2. 运行环境

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11（64 位） |
| 浏览器 | 已安装 Google Chrome（学信网验证功能需要） |
| 网络 | 需要互联网连接（AI 识别和学信网查询） |
| 权限 | 当前用户可写 `%LOCALAPPDATA%` |

首次运行时如果系统弹出 Windows Defender SmartScreen 提示，点击「更多信息」→「仍要运行」即可。

## 4. 操作流程

### 4.1 导入证书

点击工具栏 **「导入证书」** 按钮，在文件选择对话框中选取一个或多个证书文件。

**支持的文件格式：**

| 类型 | 格式 |
|------|------|
| 图片 | JPG、JPEG、PNG、BMP、WEBP |
| 文档 | PDF（需含文本层；纯扫描件请先导出为图片） |

- 支持一次选择多个文件批量导入
- 已导入的文件自动去重，不会重复添加
- 导入后文件出现在下方「待核验队列」中

### 4.2 识别证书

在队列中选中一条或多条记录，通过以下任一方式触发识别：

- 点击右侧 **「识别证书」** 按钮
- 右键菜单选择 **「识别证书」**
- 双击队列中的记录

**识别行为说明：**

| 文件类型 | 识别方式 |
|----------|----------|
| 图片文件 | 发送至 AI 视觉模型，自动纠正方向并提取字段 |
| PDF 文件 | 提取文本层内容，发送至 AI 文本模型提取字段 |

识别完成后，右侧「识别结果」区域自动填充：

- **姓名**：证书持有人姓名
- **证书编号**：电子注册号 / 证书编号
- **学校**、**专业**：自动识别（队列中显示）

> 识别结果可手动修正。如果 AI 识别有误，直接在输入框中修改即可。

### 4.3 图片方向修正

如果导入的图片方向不正确（如横置、倒置）：

- 工具会自动识别图片方向并旋转至正向
- 如果自动旋转不准确，点击预览区标题栏右侧的 **「顺转 90°」** 链接手动调整
- 每次点击顺时针旋转 90°，手动调整后不再自动覆盖

### 4.4 打开学信网验证

识别完成且姓名、证书编号均已填充后：

1. 选中队列中要验证的记录（支持 Ctrl/Shift 多选）
2. 点击 **「打开学信网验证」** 按钮

工具会自动执行以下步骤：

```
打开学信网查询页 → 填写姓名和证书编号 → 截取验证码图片 →
AI 识别验证码 → 填入验证码 → 自动点击「免费查询」
```

**多选批量验证：** 选中多条记录时，每条记录分配独立的浏览器标签页并行处理，自动错开请求避免风控。

### 4.5 人工完成扫码

验证码提交成功后，学信网页面会显示二维码。此时需要：

1. 使用手机扫描页面上的二维码
2. 在手机上完成身份验证
3. 查看并确认学历查询结果

> **注意：** 手机扫码步骤无法自动化，必须由 HR 人工完成。

### 4.6 验证码重试

如果 AI 验证码识别失败（页面提示验证码错误）：

- 工具会自动重试最多 2 次
- 自动重试仍失败时，状态显示「识别失败」或「待人工验证」
- 点击 **「重新识别验证码」** 按钮可手动重试
- 也可以直接在学信网页面上手动输入验证码并点击查询

## 5. 队列操作

### 5.1 选中与切换

- **单击**队列行：选中并在右侧预览和显示识别结果
- **Ctrl + 单击**：多选 / 取消选择
- **Shift + 单击**：范围选择

### 5.2 右键菜单

| 操作 | 说明 |
|------|------|
| 识别证书 | 对选中记录执行 AI 识别 |
| 学信网验证 | 对选中记录打开学信网并自动填写 |
| 删除证书 | 从队列中移除（不删除原始文件） |

### 5.3 移除记录

点击工具栏 **「移除当前」** 或右键选择 **「删除证书」**，从队列中移除选中记录。原始文件不受影响。

---

## 6. 状态说明

队列「状态」列显示每条记录的当前处理阶段：

| 状态 | 含义 |
|------|------|
| 待识别 | 已导入，等待执行识别 |
| 识别中 | AI 正在识别证书字段 |
| 已识别 | 识别完成，可进行学信网验证 |
| 打开中 | 正在连接浏览器并加载学信网 |
| 识别验证码中 | AI 正在识别验证码图片 |
| 已提交查询 | 验证码已自动填入并提交，请扫码 |
| 待人工验证 | 自动识别失败，需人工输入验证码 |
| 识别失败 | 验证码多次识别错误，需手动重试 |
| 打开失败 | 浏览器连接或页面加载失败 |
| 校验失败 | 姓名或证书编号格式不合法 |

## 7. 常见问题

### Q: 启动后提示「浏览器连接失败」

确保 Google Chrome 已安装且可正常运行。工具需要通过 Chrome 自动化协议控制浏览器。如果 Chrome 安装在非默认路径，可能需要手动启动 Chrome 后重试。

### Q: 图片识别结果不准确

- 确认图片清晰、文字可读
- 检查图片方向是否正确（使用「顺转 90°」调整）
- 识别结果支持手动修正，直接编辑输入框即可

### Q: PDF 无法识别

- 工具从 PDF 文本层提取文字，纯扫描件（图片型 PDF）无法提取
- 解决方案：将 PDF 导出为 JPG/PNG 图片后重新导入

### Q: 学信网页面结构变化导致填写失败

学信网改版可能导致自动填写脚本失效。此时状态显示「打开失败」，详情中会提示具体原因。请手动在学信网上完成查询。

### Q: 验证码始终识别失败

- 验证码图片模糊或干扰严重时 AI 可能无法正确识别
- 点击「重新识别验证码」重试（会刷新验证码图片）
- 也可直接在学信网页面手动输入验证码

### Q: 日志文件位置

独立 EXE 运行日志保存在：

```
%LOCALAPPDATA%\EducationCertificateTool\education_tool.log
```

遇到问题时可查看此日志定位原因。

## 8. 隐私与安全

- 证书图片/PDF 在识别时会发送至 AI 模型处理，请确保已取得候选人授权
- API Key 加密内置于程序中，不明文暴露
- 工具不会在本地持久存储证书图片，仅保留识别结果
